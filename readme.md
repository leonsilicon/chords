# Chords

These chords are my personal chords for the (unreleased) Chords app, which supports loading repositories.

# Some General Patterns

# Directions

l = Left
c = Center
r = Right

n = North / New
e = East / End
s = South / Start
w = West / Forward

t = Top
m = Middle
b = Bottom

u = Up
d = Down

a = After
o = Before

k = Back

v for Previous
x for Next

g for Go (to)

## IDEs

`f` = `F`ile/`F`ind
`p` = `P`anel (Tool Window in IntelliJ, View in VSCode)
`s` = `S`plit (Split in Intellij, Editor Group in VSCode, Pane in Warp)

## Universal Shortcuts

No need to redefine chords for certain universal shortcuts:

cmd+a - Select All
cmd+c - Copy
cmd+x - Cut
cmd+v - Paste
cmd+z - Undo
